- [Download from ThemeWagon](https://themewagon.com/themes/Materially/)

## Getting Started

1. Prerequisites:

```
node.js
```

2. Clone from Github

```
git clone https://github.com/themewagon/materially-free-react-admin-template.git
```

3. Install dependencies

```
yarn

or, npm i
```

4. Run project

```
yarn start
or, npm run start 
```
